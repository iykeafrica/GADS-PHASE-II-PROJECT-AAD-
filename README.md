# GADS-PHASE-II-PROJECT-AAD
Building a GADS Leader Board Mobile Application.

This project implements a mobile design of the GADS leader board. It displays the TOP 20 Learners in the Learning Leaders' category and the TOP 20 Learners in the Skill IQ Leaders' category.
